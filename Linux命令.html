Linux常用命令

一、基本格式

命令 [选项] [参数]

注：多个选项可以写在一起！

二、文件处理命令

  1. 目录处理命令
    1. 建立目录： mkdir -p [目录]
      • make directories -p 递归创建
    2. 切换所在目录： cd [目录]
      • change director
      • 简化操作
      • cd
      • cd ~ 进入当前用户家目录
      • cd - 进入上次目录
      • cd .. 进入上一级目录
      • cd . 进入当前目录
    3. 查询所在目录位置： pwd
      • print working director
    4. 删除文件或目录： rm -rf [文件或目录]
      • rmdir 删除空目录 （极少用）
      • -r 删除目录 -f 强制
    5. 复制命令： cp [选项] [源文件或目录] [目标目录]
      • -r 复制目录
      • -p 连带文件属性复制
      • -d 若源文件是链接文件，则复制链接属性
      • -a 相当于 -pdr
    6. 剪切或改名命令：mv [源文件或目录] [目标目录]
      • 当源文件或目录与目标目录在同一目录下时为改名
